# API Fuzzer

A simple Python tool to fuzz API endpoints using a wordlist.  
Useful for learning web security and bug bounty basics.

---

## Features

- Fuzz API endpoints using a wordlist
- Detect valid endpoints (non-404 responses)
- Simple and easy to understand
- Beginner friendly

---

## Installation

```bash
pip install -r requirements.txt
```

## Usage
```bash
python3 fuzzer.py
```
Enter target when prompted
